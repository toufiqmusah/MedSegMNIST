# MedSegMNIST - A Python Library for Easy Access to 2D/3D Medical Image Segmentation Datasets for Benchmarking and Learning Purposes

MedSegMNIST is a python library for easy access to 2D and 3D Medical Image Segmentation datasets provided in varying resolutions/sizes, for benchmarking and learning purposes. <br>
Imagine having your favourite segmentation dataset ready-to-use within your environment in 3-4 lines of code.
